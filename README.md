# Spring Cloud Sample Repo.
* Spring Cloud 를 학습하기 위한 레포지토리입니다.

### 설계

* MSA 구조
* Saga 패턴 구현



### 백엔드 (WAS)

* Spring Cloud

* Elastic search

* Apache Kafka

* Redis

  

### 프론트엔드

* Next.js : SSR / CSR 구현
* Nginx : 웹 서버



### DB

* PostgreDB
* 가능하면 DynamoDB



### 기타

* (추후) Docker & Kubernetes
